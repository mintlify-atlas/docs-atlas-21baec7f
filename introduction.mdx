## What is Tullidos SoloQ Ladder?

Tullidos SoloQ Ladder is a **custom League of Legends ladder tracker** that lets you monitor a personalized group of players with real-time rankings, daily LP tracking, and comprehensive player analytics.

Perfect for friend groups, communities, or competitive teams who want to track their progress and compete on a custom leaderboard.

## How It Works

The ladder tracker consists of two main components:

1. **Backend API** (Express.js) - Fetches data from Riot Games API, manages caching, handles rate limiting, and exposes REST endpoints
2. **Frontend App** (React + Vite) - Displays the ladder, player cards, daily highlights, and interactive animations

Players are managed through a friends list. The system automatically:
- Fetches ranked data for each player
- Calculates ladder positions based on SoloQ rank
- Tracks daily LP changes
- Detects champion pools and main roles
- Caches data to minimize API calls

<Note>
You'll need a **Riot Games API key** to use this application. Get one from the [Riot Developer Portal](https://developer.riotgames.com/).
</Note>
